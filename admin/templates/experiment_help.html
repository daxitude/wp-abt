This page allows you to create an experiment, add variations, and monitor its progress while running.

Experiment Name (r) - any descriptive name you'd like to provide for the experiment.

Goal Name (r) - any descriptive name you'd like to provide for the goal.

Goal Page (r) - the page on your site that you want to track for conversions.

Desired Confidence - e.g., 0.9 => 90% confidence that the observed results are true.

Desired Observed Effect - aka sensitivity: the change (+/-) in the conversion rate that you'd like to detect.

Status - ready, running, and completed.

Variations - the first variation created is automatically set as the base (control) variation. If you delete the base variation before starting the experiment, the next variation in the experiment will be set as the base variation. You can modify variations only before an experiment has been started.
